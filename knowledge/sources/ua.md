# Джерела: Україна 🇺🇦 (пріоритет)

> Офіційні/публічні ресурси. Доступність і URL змінюються — перевіряй на дату роботи.
> ⚠️ **Воєнний стан:** доступ до деяких реєстрів обмежено або потребує електронної
> ідентифікації (Дія/КЕП). Частина даних відкриється після скасування воєнного стану.

## Ключі-ідентифікатори

- **ЄДРПОУ** — 8-значний код юрособи (є контрольна сума; валідація в енричері `ua_company`).
- **РНОКПП / ІПН** — 10-значний податковий номер фізособи (є контрольна сума; валідація в
  `ua_person`). Персональний реквізит — обробляй лише за наявності підстави.
- **Кадастровий номер** — ідентифікатор земельної ділянки.

## API та програмний доступ (для автоматизації)

### Opendatabot API — головний інструмент автоматизації
Docs: docs.opendatabot.com. Версії **v2/v3**, формат JSON, потрібен **API-ключ**
(окремий під версію). Сервіси:

| Сервіс | Дані |
|--------|------|
| CompanyService | компанії та ФОП (за ЄДРПОУ/назвою), реєстрація, КВЕД, бенефіціари |
| CourtService | Єдиний реєстр судових рішень (ЄДРСР) |
| PhysicalPersonService | фізособи |
| PenaltyService | виконавчі провадження / Єдиний реєстр боржників (ЄРБ) |
| RealEstateService | нерухомість |
| TransportService | транспорт |
| BusinessMonitoringService | моніторинг змін (підписка на оновлення) |
| KoatuuService | довідник адмінодиниць (КОАТУУ) |

Безкоштовний відкритий доступ є до частини держреєстрів (ЄДР, ЄДРСР, ЄРБ).
Ключ кладемо у `scripts/.env` (`ODB_API_KEY`, `ODB_API_VERSION`). Енричер: `opendatabot` (UA).

### YouControl / YouScore
- **OpenData (безкоштовний тариф):** 22 реєстри, ~7 пошуків/день, моніторинг 10 компаній,
  безстроково. Гарний для ручної перевірки.
- **YouScore API** — програмна перевірка контрагентів (платно). 7+ млн компаній/ФОП із 220+ джерел:
  бенефіціари, зв'язки (адреса/телефон/керівник), фінскоринг, суди, податкові борги, санкції,
  офшори, тендери, ТМ, авто. Ключ: `YOUCONTROL_API_KEY` (за наявності).

### Інші
- **НАЗК декларації:** відкрите API — public.nazk.gov.ua/public_api (без ключа).
- **ProZorro:** відкрите API (prozorro.gov.ua) — тендери/закупівлі.

## Нотатки з практики

- ЄДРПОУ — міст між реєстрами юросіб; має контрольну суму (валідуй).
- Бенефіціари в ЄДР відкриті — Україна тут прозоріша за РФ.
- Реєстр судових рішень (reyestr) — надпотужне джерело: повні тексти, шукай за ПІБ/назвою/ЄДРПОУ.
- Під воєнний стан частина реєстрів обмежена; майнові дані часто потребують Дія/КЕП.
- Транслітерація: укр→лат (Іваненко/Ivanenko) і укр↔рос написання — тримай варіанти.
- Сверка із міжнародними санкціями — [intl.md](intl.md); РФ-аналоги — [ru.md](ru.md).
