# Джерела за країнами

Каталоги відкритих джерел організовані за країнами. Пріоритет — **Україна**.

| Країна | Файл | Статус |
|--------|------|--------|
| 🇺🇦 Україна | [ua.md](ua.md) | **пріоритет** — повне покриття |
| 🇷🇺 Росія / СНД | [ru.md](ru.md) | підтримується |
| 🌍 Міжнародні | [intl.md](intl.md) | глобальні джерела, санкції, інфраструктура |
| ➕ Нова країна | [_country-template.md](_country-template.md) | шаблон для додавання |

## Як додати країну

1. Скопіюй `_country-template.md` у `<код>.md` (ISO-2: `cz`, `pl`, `de`, `us`…).
2. Заповни таблиці офіційними джерелами (реєстри компаній, суди, майно, борги,
   санкції, держзакупівлі, PEP). Лише відкриті/законні ресурси.
3. Додай рядок у таблицю вище.
4. (Опційно) Створи енричер `scripts/enrichers/<код>_company_enr.py` за зразком
   `ua_company_enr.py` / `ru_company_enr.py` — валідація коду компанії + deep-лінки на реєстри.
5. Якщо країна стає пріоритетною — онови `CLAUDE.md`.

Заплановані до додавання (за запитом користувача): 🇨🇿 Чехія, 🇵🇱 Польща,
🇩🇪 Німеччина, 🇺🇸 США та інші.
