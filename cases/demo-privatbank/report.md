## 1. Executive Summary

Прогон через веб-платформу (`/api/enrich`) подтвердил рабочий end-to-end пайплайн:
идентификация компании → закупки → веб-инфраструктура. Все данные — из открытых
источников и публичных API, без ключей.

Ключевое: идентификатор **ЄДРПОУ 14360570 валиден**; компания крайне активна в госзакупках
(**2817 упоминаний** в ProZorro); веб-инфраструктура `privatbank.ua` вынесена на **AWS**
(Global Accelerator + Route 53), а не на собственный/локальный хостинг.

## 2. Идентификация и закупки (company)

- **ЄДРПОУ 14360570** — контрольная сумма сходится `[A1]` (`ua_company` enricher).
- **ProZorro:** 2817 упоминаний по коду; тематика — операционные закупки банка
  (оборудование АТМ, клининг по регионам, переводы, аккумуляторы ТСО) `[B2]` (ProZorro API).
- Сгенерированы deep-ссылки на ЄДР, reyestr, ЄРБ, АСВП, санкції РНБО для углубления.
- Полный профиль (директор/бенефициары/статус) — через ЄДР/Opendatabot (ключ) — оставлено как гипотеза.

## 3. Веб-инфраструктура (domain → ip)

- **NS:** `ns-1357.awsdns-41.org`, `ns-1656.awsdns-15.co.uk`, `ns-179.awsdns-22.com`,
  `ns-675.awsdns-20.net` → **AWS Route 53** `[B2]`.
- **A-записи:** `75.2.32.163`, `99.83.167.35` `[B2]`.
- **IP 75.2.32.163:** AS16509 Amazon, **AWS Global Accelerator**, гео Seattle (US) `[B2]` (ip-api).
- **Вывод:** периметр сайта на AWS; гео IP (US) — это точка присутствия CDN/Accelerator,
